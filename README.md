# 🕵️‍♂️ Mystery Script Game

An interactive Android detective game where players take on the role of an investigator solving a high-profile crime. The game combines storytelling, logic, and immersive UI/UX to challenge users through quizzes, clue analysis, and image-based investigations.

---

## 🚀 Features

- 🧩 Quiz-Based Gameplay – Solve crime scenarios through multiple-choice questions  
- 🔍 Interactive Crime Scenes – Zoomable images for detailed investigation  
- ⏱️ Countdown Timer – Time-bound clue analysis for added challenge  
- ⭐ Performance Rating System – Earn stars based on accuracy and speed  
- 🎵 Background Music & Animations – Enhanced user engagement  
- 📊 Progress Tracking – Level unlock system using SharedPreferences  

---

## 🛠️ Tech Stack

- Language: Java  
- Framework: Android SDK  
- Tools: Android Studio  
- UI: XML  
- Storage: SharedPreferences
